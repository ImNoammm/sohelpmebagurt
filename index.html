🖼️ Claude Desktop + תמונות

יש לך את ה-MCP מותקן?

מעולה! עכשיו רק תדביק את זה ל-Claude 👇

🖼️ התקנת תמיכה בתמונות

מאפשר ל-Claude לראות תרשימים ותמונות בשאלות הבגרות. עובד רק עם Claude Desktop.

1

מומלץהתקנה אוטומטית — הדבק ב-PowerShell

מוריד את הקובץ לתיקיית Documents\ClaudeMCPs, מתקין ספריות ומעדכן את ה-config אוטומטית.

irm https://raw.githubusercontent.com/ImNoammm/sohelpmebagurt/main/mcp/install.ps1 | iex
1

מומלץהתקנה אוטומטית — הדבק בטרמינל

מוריד את הקובץ לתיקיית Documents/ClaudeMCPs, מתקין ספריות ומעדכן את ה-config אוטומטית.

curl -fsSL https://raw.githubusercontent.com/ImNoammm/sohelpmebagurt/main/mcp/install.sh | bash
2

הפעל מחדש את Claude Desktop

סגור את Claude Desktop לחלוטין ופתח אותו מחדש. הכלי get_pdf_page יופיע ברשימת הכלים.

3

השתמש ב-URL של skill_mcp.md

העתק את ה-URL עם MCP מהכרטיס של המקצוע ושלח אותו ל-Claude. הוא ידע לקרוא תמונות מה-PDF אוטומטית כשיש תרשים בשאלה.

התקנה ידנית ▾

הורד את הקובץ ידנית:

⬇ הורד pdf_page_server.py

התקן ספריות:

pip install mcp pymupdf requests

הוסף ל-claude_desktop_config.json (Windows — שים לב לנתיב עם \\):

{ "mcpServers": { "sohelpmebagurt-pdf-viewer": { "command": "python", "args": ["C:\\\\Users\\\\YourName\\\\Documents\\\\ClaudeMCPs\\\\pdf_page_server.py"] } } }
