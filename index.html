תיקון לפורמט טקסט ב-Claude

הדבק את הקוד הבא ב-Console של Claude Desktop כדי שטקסט עברי יוצג מימין לשמאל:


    
    
    

      

        
1

        

          הפעל Developer Mode — צריך לעשות רק פעם אחת.

          לאחר התקנת ה-MCP, לחץ על ☰ בפינה השמאלית העליונה של Claude Desktop ← Help ← Troubleshooting ← Enable Developer Mode
        

      

      

        
2

        

          פתח שיחה חדשה. לחץ לחיצה ימנית בכל מקום בחלון ← Inspect Element ← לחץ על הכרטיסייה Console ← הדבק את הקוד.

          אם תתבקש, כתוב allow pasting ולחץ Enter, ואז הדבק שוב ולחץ Enter. 🎉
